# 06 - 页面架构

## 6.1 页面清单

|----------|------|--------|------|

| 空结果页 | - | P1 | 列表/搜索无结果时的状态页 |

- 一级页面：
  - 首页
  - Logo 生成页
  - Logo 案例列表页
  - 设计方案列表页
  - 优秀网站导航页
  - 作品集列表页
  - 作者列表页
  - 工具集合页
  - 搜索结果页
  - 关于页
  - 提交推荐页
  - 收藏页
- 二级页面：
  - Logo 案例详情页
  - 设计方案详情页
  - 网站详情页
  - 作品集详情页
  - 作者详情页
  - 工具详情页
  - 分类聚合页
  - 专题页
- 三级页面：
  - 详情页内的相关推荐落地页
  - 作者详情下的作品跳转页
  - 分类聚合页下的细分筛选结果页
  - 搜索结果页下的具体内容详情页

## 6.3 页面关系图

- 页面跳转关系：
  - 首页可跳转至 Logo 生成、案例、设计方案、导航网站、作品集、作者、工具等一级页面。
  - 各列表页可进入对应详情页，并通过标签、分类、相关推荐跳转到其他相关内容。
  - 搜索结果页可统一跳转至案例、工具、作者、网站、方案等详情页或聚合页。
  - 作者详情页与作品集详情页双向关联，形成“作者 - 作品”浏览链路。
  - 工具详情页、网站详情页可跳转至外部站点，同时保留站内相关推荐回流入口。
  - 专题页和分类聚合页承担跨模块分发功能，可连接案例、工具、网站和作者内容。
- 返回逻辑：
  - 详情页优先返回来源列表页；若用户直接进入详情页，则返回所属模块首页。
  - 外部跳转内容使用新窗口打开，用户回到站内时保留原浏览上下文。
  - 搜索与筛选条件在返回时保留，避免用户重复筛选。
  - 面包屑用于承接首页、列表页、分类页到详情页的层级回退。

## 6.4 页面布局说明

### 布局类型A
- 适用页面：首页、专题页、关于页
- 布局结构：顶部导航 + Hero 区 + 核心模块分发区 + 推荐内容区 + 页脚
- 栅格系统：桌面端 12 栏栅格，平板端 8 栏，移动端 4 栏；模块卡片支持跨栏组合展示

### 布局类型B
- 适用页面：案例列表页、工具集合页、网站导航页、作者列表页、作品集列表页
- 布局结构：顶部导航 + 页面标题区 + 筛选/排序栏 + 列表内容区 + 分页/加载更多
- 栅格系统：卡片式列表布局，桌面端 3-4 列，平板端 2 列，移动端 1 列

### 布局类型C
- 适用页面：案例详情页、工具详情页、网站详情页、作者详情页、作品集详情页、设计方案详情页
- 布局结构：顶部导航 + 详情主内容区 + 侧边信息区/辅助信息区 + 相关推荐区
- 栅格系统：桌面端双栏布局，主内容与侧边栏比例建议为 8:4；移动端改为单栏纵向布局

### 布局类型D
- 适用页面：Logo 生成页、提交推荐页
- 布局结构：顶部导航 + 操作面板区 + 结果展示区 + 说明/帮助区
- 栅格系统：桌面端左右分栏，左侧操作、右侧结果；移动端采用分段折叠布局

## 6.5 通用页面模板

- 列表页模板：
  - 包含页面标题、简要说明、筛选条件、排序方式、内容卡片列表和分页/无限加载
  - 适用于案例、工具、作者、网站、作品集等内容集合页面
- 详情页模板：
  - 包含标题信息、主视觉/预览区、核心内容说明、标签信息、相关推荐和外部跳转入口
  - 适用于案例、工具、网站、作者、作品集和设计方案详情页面
- 表单页模板：
  - 包含表单标题、字段说明、输入区域、提交按钮、校验反馈和提交成功页引导
  - 适用于 Logo 生成参数输入、资源提交推荐等交互场景
- 空白/错误页模板：
  - 包含状态说明、原因提示、推荐操作和回流入口
  - 适用于 404、无搜索结果、无收藏内容、加载失败等场景
